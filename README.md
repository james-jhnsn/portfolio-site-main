# Portfolio Website

A personal portfolio website built to showcase my software development projects, technical skills, and professional background.

## Live Demo

[View the deployed site](https://jamesjohnson.codes/)

## Features

- Responsive portfolio layout
- About section highlighting background and skills
- Project showcase with links to GitHub repositories and live demos
- Contact section
- Clean, simple UI for recruiters and hiring managers

## Built With

- HTML
- CSS
- JavaScript

## What I Learned

- Structuring a personal portfolio site
- Building responsive layouts
- Organizing project sections for readability
- Improving frontend styling and user experience
- Presenting technical work in a recruiter-friendly way
